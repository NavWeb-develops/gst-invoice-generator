# GST Invoice Generator 🇮🇳

A clean and professional GST Invoice Generator built using PHP.

## 🚀 Features

* Generate GST invoices instantly
* Supports CGST, SGST, IGST
* Auto calculation of taxes
* Indian number formatting (₹ format)
* Amount in words conversion
* Print-ready invoice layout

## 🛠 Tech Stack

* PHP
* HTML5
* CSS3
* JavaScript
